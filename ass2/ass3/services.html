A bus network topology is a network architecture in which a set of clients (nodes) are connected via a shared communication line called a bus.

A ring network is a network topology in which each node is connected to exactly two other nodes. This forms a single continuous pathway for signals through each node a RING Data travels from each node to each node, with each node handling every packet.

Star networks are one of the most common computer network topologies. In its simplest form, a star network consists of a central switch, hub or computer, which acts as a conduit to transmit messages.
